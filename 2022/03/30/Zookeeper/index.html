# zk 客户端命令

  • ls -- 查看某个目录包含的所有文件，eg：ls /
  • ls2 -- 查看某个目录包含的所有文件，与 ls 不同的是它查看到 time、version 等信息，eg： ls2 /
  • create -- 创建 znode，并设置初始内容，eg：create /test "test"
  • get -- 获取 znode 的数据，eg：get /test
  • set -- 修改 znode 内容，eg：set /test "ricky"
  • delete -- 删除 znode，eg：delete /test
  • quit -- 退出客户端
  • help -- 帮助命令
